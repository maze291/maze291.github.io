Method

A CSV-first data workflow for directional career intelligence.

Role direction

The private workflow compares analyst and analyst-adjacent roles across data, business, BI, finance, operations, product, and strategy paths.

Market focus

The private dashboard groups reviewed records by geography, role category, skills, freshness, salary availability, and confidence.

Skill planning

Repeated skill mentions are mapped into a taxonomy so learners can see which tools and concepts appear together.

What I built

A private pipeline that turns messy job-market samples into dashboard-ready views.

01

Cleaned and normalized data

Built Python and CSV workflows to standardize sampled analyst-role records into comparable fields.

02

Deduped and classified roles

Grouped likely repeated records and categorized analyst roles across practical career paths.

03

Mapped skill patterns

Tracked recurring tool, analytics, business, and finance skills in a private recommendation workflow.

04

Kept public output limited
